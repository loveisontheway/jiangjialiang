## springboot-jpa
JPA是Java Persistence API的简称，中文名Java持久层API，是JDK 5.0注解或XML描述对象－关系表的映射关系，并将运行期的实体对象持久化到数据库中。基于Spring Boot的JPA开发模板，可以达到自动创库创表。
